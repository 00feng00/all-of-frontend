## 学习路径

前端知识点多经常是很多初中级开发者所抱怨的一点，笔者和不少这阶段的开发者聊过，总结出来一个共通的点就是「信息差」。

这个「信息差」具体体现在：

- 不知道该学什么
- 不知道哪里有优秀资料可以学习
- 不知道如何搜索

这些点最终造成很多这阶段的开发者难以进阶的情况。因此笔者开了这个坑，准备把前端的学习路径具体地给大家列一下，从必备知识点到框架原理、前端工程化，并会附带笔者自己认为不错的资料帮助大家学习。
